The four axes — what shapes the surface
Dependency depth
How many other files depend on this one. core/types.h = maximum. A leaf agent file = zero. Deeper = higher on the surface = touch last.
Context cost
Lines of code + number of direct includes. Proxy for how much context an agent must load to work on this file. Smaller = lower on surface = easier to reach.
Completion delta
How many unresolved symbols this file exports that others are waiting on. High delta = blocking others = urgent = low on surface despite risk.
Churn rate
How often this file changes relative to its dependents. High churn on a high-dependency file = dangerous = push it high on surface.
Score formula — the surface height function
height(f) = (dep_depth × W₁) + (context_cost × W₂) − (completion_delta × W₃) + (churn × W₄)

ball rolls to: argmin height(f) where f is not yet complete
gaussian shape: nearby files cluster — touching auth/token.c pulls auth/hash.c into the valley too
