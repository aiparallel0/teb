teb2 file schema with strict I/O contracts: max 2 inputs, 1 output per file, shown as typed ports on each module

Interface laws — enforced at compile time
1.Every file exposes max 2 inputs, exactly 1 output. No exceptions. If you need 3 inputs, the file has not been decomposed enough — split it.
2.Input = a typed struct passed by pointer or a primitive scalar. Never a raw pointer to an untyped buffer. Never a global.
3.Output = return value only. Never a side-channel write through a global. Error state is encoded in the return type, not a separate errno-style global.
4.types.h and errors.h are exempt — they are vocabulary, not interfaces.
