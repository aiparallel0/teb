# TEB Tutorials

Step-by-step tutorials to help you get the most out of TEB. Each tutorial
includes written instructions and a companion video (links below).

---

## Beginner Tutorials

### Tutorial 1 – Your First Goal

Learn how to create a goal, add tasks, and track progress.

- **Duration**: ~5 min
- **Video**: [Watch on YouTube](#) <!-- TODO: add link when published -->
- **Steps**:
  1. Log in to TEB.
  2. Click **New Goal** and enter a title.
  3. Add 3–5 tasks inside the goal.
  4. Mark tasks as done and watch the progress ring fill up.

### Tutorial 2 – Using the Kanban Board

Visualise your workflow with drag-and-drop columns.

- **Duration**: ~5 min
- **Video**: [Watch on YouTube](#)
- **Steps**:
  1. Navigate to **Kanban** from the sidebar.
  2. Drag a task from "To Do" to "In Progress".
  3. Customise column names in settings.

### Tutorial 3 – Setting Up Notifications

Never miss a deadline with email and push notifications.

- **Duration**: ~4 min
- **Video**: [Watch on YouTube](#)
- **Steps**:
  1. Go to **Settings → Notifications**.
  2. Enable email and/or push notifications.
  3. Choose which events you want to be notified about.

## Intermediate Tutorials

### Tutorial 4 – AI Decomposition

Let the AI break down complex goals into actionable tasks.

- **Duration**: ~7 min
- **Video**: [Watch on YouTube](#)
- **Steps**:
  1. Create a goal with a detailed description.
  2. Click **AI Decompose**.
  3. Review, edit, and accept the suggested tasks.

### Tutorial 5 – Collaboration & Workspaces

Work together with your team in shared workspaces.

- **Duration**: ~8 min
- **Video**: [Watch on YouTube](#)
- **Steps**:
  1. Create a workspace from the admin panel.
  2. Invite members via email.
  3. Share goals within the workspace.

### Tutorial 6 – Webhooks & Integrations

Connect TEB to external tools like Slack and GitHub.

- **Duration**: ~10 min
- **Video**: [Watch on YouTube](#)
- **Steps**:
  1. Go to **Integrations** in settings.
  2. Add a new webhook URL.
  3. Select events to trigger the webhook.
  4. Test with a sample event.

## Advanced Tutorials

### Tutorial 7 – Building a Plugin

Extend TEB with a custom plugin.

- **Duration**: ~15 min
- **Video**: [Watch on YouTube](#)
- **Steps**:
  1. Read the [Plugin Guide](plugin-guide.md).
  2. Scaffold a plugin with the manifest template.
  3. Add a custom view.
  4. Test and publish to the marketplace.

### Tutorial 8 – Browser Automation

Automate web interactions as part of your execution pipeline.

- **Duration**: ~12 min
- **Video**: [Watch on YouTube](#)
- **Steps**:
  1. Create a task with browser automation enabled.
  2. Define actions (navigate, click, type).
  3. Run the execution pipeline.
  4. Review screenshots and extracted data.

### Tutorial 9 – Agent Orchestration

Set up multi-agent flows for complex automation.

- **Duration**: ~15 min
- **Video**: [Watch on YouTube](#)
- **Steps**:
  1. Create agent schedules.
  2. Define an agent flow with multiple steps.
  3. Configure handoffs between agents.
  4. Monitor execution in the agent dashboard.
